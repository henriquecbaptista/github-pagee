Portfólio profissional desenvolvido em HTML, CSS e JavaScript puro.  
Migrado do Google Sites para GitHub Pages.

## Estrutura

```
portfolio/
├── index.html          # Página inicial
├── sobre.html          # Sobre mim
├── projeto.html        # Projeto TCC
└── assets/
    ├── style.css       # Estilos
    └── main.js         # Scripts
```

## Como publicar no GitHub Pages

1. Crie um repositório no GitHub (ex: `portfolio`)
2. Faça upload de todos os arquivos desta pasta
3. Vá em **Settings → Pages**
4. Em **Source**, selecione a branch `main` e a pasta `/ (root)`
5. Clique em **Save**

Seu site ficará disponível em:  
`https://<seu-usuario>.github.io/portfolio/`
